Vector-valued Functions

Jacobian Matrix

The Jacobian matrix is the matrix of partial derivatives. Given a function f =( f 1 ( x), ( f 2 ( x), ..., ( f m ( x)) , the Jacobian matrix would be J = ( ∂ f 1 ∂ x 1 ... ∂ f 1 ∂ x n ... ⋱ ... ∂ f m ∂ x n ... ∂ f m ∂ x n )

Alternative Notation

The Jacobian matrix may be denoted with several different symbols. Some are listed below:

  • J
  • J f
  • D f
  • ∇ f
Linear Approximations

For a multivariate function ( ℝ n → ℝ m ) the Jacobian matrix gives the best linear approximnation to f at x : f( x + h v) ≈ f( x) + J f ( x)( h v)

Note:

  • x, v, h v are n -dimensional vectors
  • J f ( x) is an m by n matrix
  • f( x), J f ( x)( h v) are m -dimensional vectors.
