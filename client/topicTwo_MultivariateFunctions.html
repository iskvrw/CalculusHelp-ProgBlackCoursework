Multivariate Functions

Partial Derivatives

For a function with more than one input, you must choose one of those inputs to differentiate with respect to, meaning that you can only "partially" differentiate for a multivariate function.

For a function f( x, y) , The partial derivative with respect to x at a point ( x 0 , y 0 ) is: ∂ f ∂ x ( x 0 , y 0 ) = lim x → x 0 f (x, y 0 ) - f ( x 0 , y 0 ) x - x 0

Example

Given f( x, y) = 2 x 2 y 3 - x y + 2 x - 3 y + 5 , The partial derivative of f wrt to y is: ∂ f ∂ x = 6 x y 2 - x - 3

Alternative Notation

You may also see f x used when referring to partial derivatives, such as ∂ f ∂ x .

Directional Derivatives

Consider a point x 0 =( x 0 , y 0 ) and a vector v = ( v x v y ) . Then, the directional derivative of f in direction v is: ∇ v f = v . ( f x f y )

The symbol ∇ f is a vector pointing in the direction of greatest rate of increase of f.
