Univariate Functions: Smoothness and Higher Order Derivatives

The k th derivative of a function f is denotated as f (k)

A continuous function is in class C 0

A function that's continuous and has a continuous derivative is in class C 1

More generally, a function f such that its derivatives f (1) , f (2) ,..., f (k) exist and are continuous is in class C k .

A function can be called smooth when it is continuous and all its derivatives also exist and are continuous. Such a function is in class C ∞ .

For example, polynomials, trigonometric functions sin and cos are smooth.

Multivariate Funcitons: Smoothness and Higher Order Derivatives

A multivariate function f is smooth if it is continuous and all its partial derivatives of all orders exist and are continuous.

Bivariate Extrema

To identify a minimum or maximum given a bivariate function, we must analyse the Hessian matrix.

The Hessian matrix is a square matrix containing all second order derivatives of a bivariate function.

H f = ( ∂ f 2 ∂ x 2 ∂ f 2 ∂ x ∂ y ∂ f 2 ∂ y ∂ x ∂ f 2 ∂ y 2 )

Second derivative test for extrema

Suppose f( x, y) ∈ C 2 and f x ( x 0 , y 0 ) = 0 = f y ( x 0 , y 0 ) , then

  • ( x 0 , y 0 ) is a local maximum if f xx f yy - f 2 xx > 0 and f xx < 0 at ( x 0 , y 0 )
  • ( x 0 , y 0 ) is a local minimum if f xx f yy - f 2 xx > 0 and f xx > 0 at ( x 0 , y 0 )
  • ( x 0 , y 0 ) is a local saddle point if f xx f yy - f 2 xx < 0 at ( x 0 , y 0 )
  • If f xx f yy - f 2 xx = 0 then the test is inconclusive. Higher order derivatives must be analysed.
