Differentiation

The Derivative

The first derivative is often referred to as the gradient, or "the slope", of a line

The Gradient

At a point \(x_0\) ,it can be approximated by:

\(\frac{f(x) - f(x_0)}{x - x_0}\)

This is more commonly used as the limit of a variable h (the distance from the point) as it approaches \(0\):

\(\lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\)

Notation

The symbols used for the derivative include: \(f'(x)\) and \(\frac{df}{dx}\)

Basic Derivatives

Below are some basic derivatives which underpin differentiation.

If \(f(x)=c\) for some constant \(c\) ,then \(f'(x)=c\), \(\forall x\)

If \(f(x)=x\), then \(f'(x)=1\).

If \(f(x)=x^n\), then \(f'(x)=nx^{n-1}\)

If \(f(x)= \sin{kx} \),then \(f'(x)=k \cos{kx} \)

If \(f(x)=k \cos{kx} \), then \(f'(x)=-k \sin{kx} \)

If \(f(x)=g(x)+h(x) \), then \(f'(x)=g'(x)+h'(x) \)

Nice Functions

These are functions which are continuous \(\forall x\) so their first derivative can be calculated for all \(\forall x\).

Not so Nice Function

An example of this would be \(f(x)= \tan{x}\)

The function cannot be evaluated at certain points, namely \(x=\frac{k \pi}{2}\) for \(k \in \mathbb{N} \)as it tends to infinity when approaching from either side. Thus, its derivative cannot be calculate at those points and the function is not nice.

Continuity

For a function \(f \) to be continuous at a point \(x=a\), then

\(\lim_{h \to a^+} f(x) = \lim_{h \to a^-} f(x) = L \)

and

\(f(x)=L \)

for some constant L.

General Rules

Below are some rules used for differentiation, where \(f(x)\) and \(g(x) \) are differentiable at \(x_0 \)

Product Rule

\(\frac{d}{dx} (f(x)g(x)) = f'(x)g(x) + f(x)g'(x) \)

Quotient Rule

\(\frac{d}{dx} (\frac{f(x)}{g(x)}) = \frac{f'()g(x)-f(x)g'(x)}{(g(x))^2} \)

Chain Rule

\(f'(x) = g'(x) + h'(x) \)

Extrema

These are points where a function reaches a flat point.

The two types of extrema are minima and maxima. Minima are low points and maxima are high points

Determining Extrema

Let \(f(x) \) be defined over \([a,b]\). A point \(x_0 \in [a, b]\) is:

  • An absolute maximum if \(f(x_0) \ge f(x), \forall x \in [a,b] \)
  • An absolute minimum if An absolute maximum if \(f(x_0) \le f(x), \forall x \in [a,b] \)
  • An local maximum if \(\exists \delta \) such that \(f(x) \ge f(x+h), \mid h \mid \lt \delta \)
  • An local minimum if \(\exists \delta \) such that \(f(x) \le f(x+h), \mid h \mid \lt \delta \)
