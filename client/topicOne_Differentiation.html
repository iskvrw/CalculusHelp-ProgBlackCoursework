The Derivative

The first derivative is often referred to as the gradient, or "the slope", of a line

The Gradient

At a point x 0 ,it can be approximated by:

f( x) - f( x 0 ) x - x 0 This is more commonly used as the limit of a variable h (the distance from the point) as it approaches 0: lim h → 0 f( x + h) - f( x) h

Notation

The symbols used for the derivative include: f '( x) and d f d x

Basic Derivatives

Below are some basic derivatives which underpin differentiation.

If f( x) = c for some constant c ,then f '( x) = c ∀ x

If f( x) = x ,then f '( x) = 1 .

If f( x) = x n ,then f '( x) = n x n - 1

If f( x) = sin( k x) ,then f '( x) = k cos( k x)

If f( x) = cos( k x) ,then f '( x) = - k sin( k x)

If f( x) = g( x) + h( x) ,then f '( x) = g '( x) + h '( x)

Nice Functions

These are functions which are continuous ∀ x so their first derivative can be calculated for all x.

Not so Nice Function

An example of this would be f( x) = tan( x)

The function cannot be evaluated at certain points, namely x = k π 2 for k ∈ ℕ as it tends to infinity when approaching from either side. Thus, its derivative cannot be calculate at those points and the function is not nice.

Continuity

For a function f to be continuous at a point x=a, then lim h → a + f( x ) = lim h → a - f( x ) = L and f( x ) = L for some constant L.

Extrema

These are points where a function reaches a flat point.

The two types of extrema are minima and maxima. Minima are low points and maxima are high points

Determining Extrema

Let f(x) be defined over [a,b] .

A point x 0 ∈ [a,b] is:

  • An absolute maximum if f( x 0 ) ≥ f( x) ∀ x ∈ [a,b]
  • An absolute minimum if f( x 0 ) ≤ f( x) ∀ x ∈ [a,b]
  • An local maximum if ∃ k>0 , f( x 0 ) ≥ f( x+h) ∀ | h | < k
  • An local minimum if ∃ k>0 , f( x 0 ) ≤ f( x+h) ∀ | h | < k
